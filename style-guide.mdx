<Info>
  Text colors are guidelines — certain cases require deviations. An example is `text` on colored backgrounds. Legibility always comes first, so please use your own judgement.
</Info>

## Background colors

| Function | Color(s) |
| --- | --- |
| Background pane | Base |
| Secondary panes | Crust, Mantle |
| Surface elements | Surface 0, Surface 1, Surface 2 |
| Overlays | Overlay 0, Overlay 1, Overlay 2 |

## Typography

| Function | Color(s) |
| --- | --- |
| Body copy | Text |
| Main headline | Text |
| Sub-headlines, labels | Subtext 0, Subtext 1 |
| Subtle | Overlay 1 |
| On accent | Base |
| Links, URLs | Blue |
| Tags, pills | Blue |

## Status colors

| Status | Color |
| --- | --- |
| Success | Green |
| Warnings | Yellow |
| Errors | Red |

## Interactive elements

| Element | Color |
| --- | --- |
| Selection background | Overlay 2 at **20%–30% opacity** |
| Cursor | Rosewater |
