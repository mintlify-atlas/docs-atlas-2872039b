  </Tab>

## Userstyles

[Userstyles](https://github.com/catppuccin/userstyles) apply Catppuccin to websites using CSS overrides, compatible with extensions like Stylus. The repository covers hundreds of sites including GitHub, Discord, Reddit, YouTube, and many more. Browse the full list in the [`catppuccin/userstyles`](https://github.com/catppuccin/userstyles) repository.
