---
title: "CLI tools"
description: "Catppuccin themes for command-line tools, TUI applications, and shell utilities."
---

Catppuccin is available for 23+ CLI and TUI tools. Each port configures colors, prompts, and UI elements to match a Catppuccin flavor.

- [Asciinema](https://github.com/catppuccin/asciinema) — terminal session recorder and player
- [atuin](https://github.com/catppuccin/atuin) — shell history sync and search
- [bat](https://github.com/catppuccin/bat) — `cat` clone with syntax highlighting
- [bottom](https://github.com/catppuccin/bottom) — graphical process and system monitor
- [btop++](https://github.com/catppuccin/btop) — resource monitor with an interactive TUI
- [delta](https://github.com/catppuccin/delta) — syntax-highlighting pager for `git diff`
- [fish shell](https://github.com/catppuccin/fish) — fish shell color scheme
- [fzf](https://github.com/catppuccin/fzf) — command-line fuzzy finder
- [GitUI](https://github.com/catppuccin/gitui) — terminal-based Git UI
- [k9s](https://github.com/catppuccin/k9s) — Kubernetes CLI dashboard
- [lazygit](https://github.com/catppuccin/lazygit) — terminal UI for Git commands
- [LSD](https://github.com/catppuccin/lsd) — `ls` replacement with icons and color
- [Newsboat](https://github.com/catppuccin/newsboat) — RSS/Atom feed reader for the terminal
- [Nushell](https://github.com/catppuccin/nushell) — Nushell color theme
- [OhMyREPL](https://github.com/catppuccin/ohmyrepl) — Julia REPL syntax highlighting
- [Process Compose](https://github.com/catppuccin/process-compose) — TUI process manager
- [sc-im](https://github.com/catppuccin/sc-im) — terminal spreadsheet editor
- [skim](https://github.com/catppuccin/skim) — fuzzy finder written in Rust
- [Starship](https://github.com/catppuccin/starship) — cross-shell prompt
- [television](https://github.com/catppuccin/television) — fuzzy finder TUI
- [tmux](https://github.com/catppuccin/tmux) — terminal multiplexer theme and status bar
- [vtop](https://github.com/catppuccin/vtop) — visual process monitor
- [Zellij](https://github.com/catppuccin/zellij) — terminal workspace and multiplexer

<Note>
  Don't see your tool? You can [create a port](/port-creation) and submit it to the organization.
</Note>
