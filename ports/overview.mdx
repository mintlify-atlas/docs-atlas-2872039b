A **port** is a Catppuccin theme adapted for a specific application or tool. Each port maps the Catppuccin palette to the theming system of its target app, so you get consistent, soothing colors everywhere you work.

The community maintains 300+ ports spanning code editors, terminals, browsers, libraries, CLI tools, desktop environments, and beyond.

## Userstyles

[Userstyles](https://github.com/catppuccin/userstyles) are a special category of port that apply Catppuccin colors to websites using CSS overrides. They cover hundreds of sites — from GitHub and Discord to search engines, wikis, and social networks — and are managed in the dedicated `catppuccin/userstyles` repository.

## Add a new port

Want to bring Catppuccin to an app that isn't covered yet? Follow the [port creation guide](/port-creation) to learn how to build and submit a port.
