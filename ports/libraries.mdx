Catppuccin provides official palette libraries for a range of programming languages and frameworks. Each library exposes all four flavors and their 26 colors in idiomatic types and structures for that language, so you can use the palette programmatically in your own projects.

## npm packages

The [`@catppuccin`](https://www.npmjs.com/org/catppuccin) npm organization publishes official JavaScript and TypeScript palette packages. The core package is [`@catppuccin/palette`](https://github.com/catppuccin/palette), which provides all four flavors and their colors as typed TypeScript objects suitable for use in any JS/TS project or framework.
