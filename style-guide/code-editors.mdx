## Language defaults

These are the standard syntax token color assignments. Apply them as your baseline when building an editor theme.

| Syntax | Color |
| --- | --- |
| Keyword | Mauve |
| Strings | Green |
| Symbols, Atoms | Red |
| Escape sequences, Regex | Pink |
| Comments | Overlay 2 |
| Constants, Numbers | Peach |
| Operators | Sky |
| Braces, Delimiters | Overlay 2 |
| Methods, Functions | Blue |
| Parameters | Maroon |
| Builtins | Red |
| Classes, Interfaces, Annotations, Metadata, Enums, Types | Yellow |
| Enum variants | Teal |
| Property (e.g. JSON keys) | Blue |
| Attributes (e.g. XML-style attributes) | Yellow |
| Macros | Rosewater |

### Parameters and local variables

These use a 60% mix of Text with the corresponding accent color to keep them visually distinct but not overwhelming.

| Level | Color |
| --- | --- |
| Color #1 | 60% Text / Red mix |
| Color #2 | 60% Text / Yellow mix |
| Color #3 | 60% Text / Green mix |
| Color #4 | 60% Text / Teal mix |
| Color #5 | 60% Text / Blue mix |
| Color #6 | 60% Text / Mauve mix |

## Diff and merge

| Function | Color |
| --- | --- |
| Diff header | Blue |
| Index metadata | Overlay 2 |
| File path markers | Pink |
| Hunk header | Peach |
| Changed text background | Blue at **10%–20% opacity** |
| Changed line background | Blue at **15%–25% opacity** |
| Inserted text background | Green at **10%–20% opacity** |
| Inserted line background | Green at **15%–25% opacity** |
| Removed text background | Red at **10%–20% opacity** |
| Removed line background | Red at **15%–25% opacity** |
